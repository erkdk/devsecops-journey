#### Problem:
- Familiarize yourself with the commands top, ps, and pstree that are used to monitor processes
running in the system. Learn how to use the man pages for these tools, e.g., by typing man ps in
a terminal. You can press ’h’ when inside a man page to get useful keyboard shortcuts to help you
effectively navigate the page.

- Understand the /proc filesystem in Linux. The /proc file system is a mechanism provided by
Linux for the kernel to report information about the system and processes to users. The /proc file
system is nicely documented in the proc man page at man proc. Understand the system-wide
proc files such as meminfo, cpuinfo and stat, as well as per-process files like status.
